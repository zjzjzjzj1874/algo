# 算法

## 滑动窗口
1. 最长子字符串

   滑动窗口的思想：两个指针+hash表；俩起始位置都相同，然后快指针每走过一个元素，尝试把元素加到hash表中，
   如果不存在则添加到hash表，如果存在，则从慢指针开始依次删，删一个再尝试加一个，直到加入成功，继续走快指针；
